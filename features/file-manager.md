# File Manager

The File Manager helps users organize and manage files within Ekgrona.

---

## Features

- Upload files
- Download files
- Create folders
- Move files
- Rename files
- Delete files
- Search files

---

## Organization

Files can be grouped into folders for better organization.

---

## Search

Quickly locate files using search functionality.

---

## Supported Operations

- Create
- Read
- Update
- Delete
