# Dashboard

The Dashboard serves as the central hub of the Ekgrona platform.

---

## Purpose

Provide quick access to:

- Recent activity
- Files
- Tasks
- Installed applications
- Utility tools

---

## Key Features

### Overview Widgets

Display important information at a glance.

### Quick Actions

Perform common actions directly from the dashboard.

### Recent Activity

Monitor recent platform activity and updates.

### Personalization

Customize dashboard layout and widgets.
