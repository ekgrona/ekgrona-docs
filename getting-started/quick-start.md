# Quick Start
This guide helps you get started with Ekgrona in just a few minutes.

---

## Step 1: Create an Account
Visit the [Registration Page](https://www.ekgrona.com/signup) and create your account using a valid email address and a secure password.
After registration, you may be required to verify your email address before accessing all platform features.

---

## Step 2: Sign In
[Log In](https://www.ekgrona.com/signin) using your email address and password.
After successful authentication, you will be redirected to your workspace dashboard.

---

## Step 3: Explore the Home Screen
In the Home Screen has the 4 Sections: Sidebar, Taskbar, Profile Menu and Main Interface.

From the Sidebar, you can:
* Toggle Sidebar fully and collapsed by clicking the Ekgrona Icon
* Access the Dashboard
* Access the Files
* Access the Tasks
* Access the Tools
* Access the Applications & Plugins
* Logout, access the Customer Support and Feedback Page

From the Taskbar, you can:
* Access the Searchbar
* See the tasks
* Manage the tasks

From the Profile Menu, you can:
* Access the Profile Page
* Access the Notifications
* Access the Settings

From the Main Interface, you can:
* Access the functionailty of each features
* Manage your productivity

---

## Step 4: Explore the Dashboard
The Dashboard serves as the central hub of Ekgrona.

From the dashboard, you can:
* See your Profile Card
* View your Files
* View recent activity
* Manage tasks
* Launch applications
* Use utility tools

---

## Step 5: Organize Your Files
Open the File Manager to begin organizing your files.

Common actions include:
* Creating folders
* Uploading files
* Renaming items
* Moving files
* Deleting files

Keeping files organized improves productivity and makes information easier to locate.

---

## Step 6: Create Your First Task
Open the Task Manager and create a task.

You can:
* Add a title
* Set a description
* Assign a priority
* Set a due date
* Track progress

---

## Step 7: Explore Applications
Applications extend the functionality of your workspace.
Browse available applications and install those that fit your workflow.
Installed applications can be accessed directly from your dashboard.

---

## Step 8: Use Utility Tools
Utility Tools provide quick access to common productivity features.
Available tools may vary as the platform evolves.

---
