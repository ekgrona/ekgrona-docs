# Quick Start

This guide helps you get started with Ekgrona in just a few minutes.

---

# Step 1: Create an Account

Visit the registration page and create your account using a valid email address and a secure password.

After registration, you may be required to verify your email address before accessing all platform features.

---

# Step 2: Sign In

Log in using your email address and password.

After successful authentication, you will be redirected to your workspace dashboard.

---

# Step 3: Explore the Dashboard

The Dashboard serves as the central hub of Ekgrona.

From the dashboard, you can:

* Access your files
* Manage tasks
* Launch applications
* Use utility tools
* View recent activity

---

# Step 4: Organize Your Files

Open the File Manager to begin organizing your files.

Common actions include:

* Creating folders
* Uploading files
* Renaming items
* Moving files
* Deleting files

Keeping files organized improves productivity and makes information easier to locate.

---

# Step 5: Create Your First Task

Open the Task Manager and create a task.

You can:

* Add a title
* Set a description
* Assign a priority
* Set a due date
* Track progress

---

# Step 6: Explore Applications

Applications extend the functionality of your workspace.

Browse available applications and install those that fit your workflow.

Installed applications can be accessed directly from your dashboard.

---

# Step 7: Use Utility Tools

Utility Tools provide quick access to common productivity features.

Available tools may vary as the platform evolves.

---
